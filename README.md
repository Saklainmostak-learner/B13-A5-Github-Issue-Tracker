1. What is the difference between var, let, and const?

Answer: var, let, and const are used to declare variables.
var is the older way of declaring variables. We can redeclare and update a variable by using var, which can make a bug in code.so now a days we skip it for variable declaration. 
let is the inline block-scope of variable declaration. we cann't redeclare the value, but we can update it's valu while using let for variable declaration. 
const is also block-scoped like let, but we can't reassign a value after declared by const.

2. What is the spread operator (...)?
   
Answer: The spread operator is used to expand elements of an array or properties of an object, and it helps copy or combine data easily.

3.What is the difference between map(), filter(), and forEach()?

Answer: map() transforming every element of the original array and creating a new array.
filter() contains only the matched element and condition, then creates a new array.
forEach() does not return a new array but simply loops through each element of an array.
